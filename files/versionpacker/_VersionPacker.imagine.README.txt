Packer utility for GoldenGATE Document Editor including Configurations

This utility packs GoldenGATE Document Editor editor core components together
with a Configuration of your choice into ZIP files, which it deposits in the
"_Zips" sub folder of the folder you run it in. The latter should be the root
folder of your local GioldenGATE Editor installation.
The main use is to pack up ready-to-use versions of GoldenGATE Editor, so you
can distribute them to your users more easily.



REQUIRED JAVA VERSION

Sun (r) Java Runtime Environment 1.5.0 or higher



STARTING THE PROGRAM

Simply double-click the JAR
